Java 按位运算符

Posted on 2019-03-12

Java 中整数为 int 类型，也就是 32 位。按位运算符是将操作数当做 32 位的二进制码。同时，Java 中使用补码来表示整数，也用补码来进行二进制码的数学运算。例如：

按位与（&）

只有两个操作数的二进制码相同的位都是 1 时，结果为 1，否则为 0。例如：2 & 3 = 2

按位或（|）

只有两个操作数的二进制码相同的位至少有一个 1 时，结果为 1，否则为 0。例如：2 & 3 = 3

按位异或（^）

只有两个操作数的二进制码相同的位有且仅有一个 1 时，结果为 1，否则为 0。例如：2 ^ 3 = 1

按位非（~）

将操作数的二进制码按位取反，即 0 变成 1，1 变成 0。例如：~ 2 = -3

左移（<<）

将操作数的二进制码向左移 n 位，右边用 0 填充，符号位保持不变。运算中我们需要把补码转到原码，才能得到它的真值。例如：-2 << 1 = -4

有符号右移（>>）

将操作数的二进制码向右移 n 位，丢弃被移出的位，符号位保持不变。例如：2 >> 1 = 1

无符号右移（>>>）

将操作数的二进制码向右移 n 位，丢弃被移出的位，左边用 0 填充。例如：-2 >> 1 = 2147483647
