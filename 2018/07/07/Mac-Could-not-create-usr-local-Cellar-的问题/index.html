Posted on 2018-07-07

最近需要安装 jq,理所当然就使用了如下命令。

但是却没有直接安装成功，提示了如下错误信息。

之前也使用 brew 安装其他软件包，没有遇到这种错误。但是它既然提示我没有权限，我便给自己的账号加上权限准备再试试。

发现还是提示了错误。

网上找了一下，发现是 os x 10.11 开始，系统使用了 Rootless 机制，目的为了防止恶意软件修改 Mac 上受保护的的文件和文件夹。其中包括了 /System, /usr, /bin, /sbin 以及 OS X 预装的应用。详情可参见苹果官方文档。

准备关掉 Rootless 来安装我需要的东西。重启 Mac,重启的同时按住 Command + R，进入恢复系统。在顶部菜单 实用工具 里选择 Terminal，输入如下命令。
