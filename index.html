Git 常用命令

Posted on 2018-12-08

git config

设置/显示仓库配置/全局配置，如果仓库配置未进行设置，会默认使用全局配置

  • --add 新增一行配置
  • --get 获取指定 Key 的值，如果不存在这个 Key 返回错误
  • -l 显示所有的配置/键值对
  • --global 全局配置模式，可与其他参数配合使用

比如，设置全局用户名，即 commit 显示的名字，如果仓库没有特别指定，会默认使用全局配置
git config --global --add user.name <username>

git clean

移除本地未跟踪的文件

  • -f 移除本地所有未跟踪的文件
  • -d 移除本地所有未跟踪的文件夹
  • -n 仅显示会被移除的文件，不会真的移除文件

git reset

重置当前的 HEAD 为指定的状态

  • --hard 丢弃所有已跟踪的文件从上次 commit 的改动

git status

显示工作树的状态

  • -s 以简短格式显示信息
  • --long 以详细格式显示信息，默认值

git commit

记录仓库的改动

  • -m 使用输入的内容作为提交的信息

git push

把本地的 commit 推到远程仓库，并更新远程仓库
